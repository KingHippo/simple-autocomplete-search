Jquery Autocomplete Search
== Description ==
A plugin using Jquery for autocomplete search function
== Installation ==
1. Download the plugin.
1. Upload the jquery-autocomplete-search folder to your site's wp-content/plugins directory.
1. Activate the plugin in the WordPress Admin Panel.
1. Upload a CSV file to your database and modify create_table function to match your database rows.
1. Add the city-pop-search shortcode to your page to implement search box.
